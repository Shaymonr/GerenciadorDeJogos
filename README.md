# GerenciadorDeJogos

Projeto de WebApp para o gerenciamento de jogos, criado durante as aulas do curso de .NetCore 6 para fixação do conteúdo.

## Ferramentas

- VisualStudio 2022 (Desenvolvimento)
- GitHub (Versionamento)

## Preview

![preview]([./assets/preview.png](https://gerenciadordejogos.onrender.com))

## Funcionalidades

- CRUD para gerenciar a lista de jogos.
- Organizar jogos em diferentes categorias e estados.
  
## Tecnologias

- HTML
- CSS
- JavaScript
- AspNet Core 8
- Bootstrap 5.3

## Como executar

```bash
git clone https://github.com/Shaymonr/GerenciadorDeJogos/
```

Abra o arquivo `index.html`.
